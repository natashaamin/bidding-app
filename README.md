# umi project

## Getting Started

## To Run Client Side

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```


## To Run Server Side

Navigate to server folder

```bash
$ cd server
```

Install dependencies, 

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```
